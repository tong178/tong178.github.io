用hexo搭建属于自己的博客

最近不想学习，想着不如做些自己感兴趣的事，看着之前挂脚本用过之后闲置的服务器，心血来潮，于是有了这个博客。当然做博客不一定要服务器，github注册一个账户就行。（我初学者大佬勿喷）
以下演示在Windows系统下创建博客并免费在GitHub上部署的教程

1 首先下载一个node.js

这是JavaScript的库和JavaScript的运行器
node.js下载链接

下载完后一步一步安装，安装完后有一个nodejs本身还有一个npm包资源管理器
安装完后用管理员身份打开cmd窗口输入 node -v ;npm -v 来查看安装版本来验证是否安装成功

2 用npm安装cnpm

我们要用之前安装的npm安装hexo框架但是npm下载太慢，可以先用npm安装cnpm再用cnpm来安装hexo框架。
输入cnpm安装命令命令：
npm install -g cnpm –registry=https://registry.npm.taobao.org

同样通过 cnpm -v 来查看安装版本是否安装成功

3 用cnpm安装hexo博客框架

输入命令：cnpm install -g hexo-cli

同样用 hexo -v查看是否安装成功

4用hexo框架生成博客

首先在node.js的安装目录下创建一个blog文件夹用来存放等下生成的博客源文件

在cmd窗口中进入blog文件夹下用命令 hexo init 来生成博客源文件

安装完成后blog里会生成这些文件

使用命令 hexo s 来启动博客

复制给的地址在浏览器里打开

一个博客就做好了但是博客不能只本地预览，接下来上传到GitHub上

5上传github

首先要先下载git部署插件
输入命令
cnpm install –save hexo deployer -git

然后登入GitHub创建一个库，库的名字必须是用户名.github.io


接下来在blog文件下修改 _config.yml文件里deploy的属性

deploy:
type: ‘git’
repo: 你的GitHub的库的地址
branch: master

最后用命令 hexo d 部署到远端

部署成功

如果部署失败

那么依次执行
hexo clean
hexo g
hexo d

6补充

常用hexo命令：
hexo clean 清除
hexo g 重新生成文件
hexo d 部署远端
hexo s 部署本地
